## 테스트 시나리오

### 시나리오 1: 여러 모달 동시 활성화
1. 위의 예시 중 여러 개를 동시에 활성화
2. Priority가 높은 순서대로 표시되는지 확인
3. 한 모달을 닫으면 다음 모달이 표시되는지 확인

### 시나리오 2: 이미지 로딩 실패
1. 잘못된 이미지 URL 입력 (예: `https://invalid-url.com/image.jpg`)
2. 이미지 로딩 실패 시 UI가 깨지지 않는지 확인
3. 이미지 없이 텍스트만 표시되는지 확인

### 시나리오 3: 긴 텍스트 처리
1. 긴 제목과 설명이 있는 모달 생성
2. 텍스트가 잘리지 않고 스크롤되는지 확인
3. 모달 높이가 적절하게 조정되는지 확인

### 시나리오 4: 날짜 범위 테스트
1. 시작일이 미래인 모달 생성 → 표시되지 않아야 함
2. 종료일이 과거인 모달 생성 → 표시되지 않아야 함
3. 현재 날짜 범위 내 모달만 표시되는지 확인

## 참고사항

1. **Template 선택 가이드**:
   - `image_top`: 이미지가 있는 경우 사용
   - `no_image`: 이미지가 없는 경우 사용
   - `product_spotlight`: 상품 강조 모달
   - `mission_card`: 미션 전용 모달

2. **Priority**: 숫자가 클수록 우선순위가 높습니다. 동시에 여러 모달이 활성화되면 Priority가 높은 것부터 표시됩니다.

3. **날짜 형식**: 관리자 페이지에서 `datetime-local` 형식으로 입력하세요. 예: `2026-01-15T00:00`

4. **이미지 URL**: 
   - Unsplash 등 무료 이미지 서비스 활용 가능
   - 이미지가 없어도 모달은 정상 작동합니다
   - 이미지가 있으면 `image_top` 템플릿 사용 권장

5. **CTA 딥링크**: 
   - `/home`, `/benefits`, `/more` 등 앱 내 경로
   - 실제 라우팅 경로와 일치해야 합니다

6. **테스트 순서**:
   - 먼저 하나씩 생성해서 각각 테스트
   - 그 다음 여러 개 동시 활성화해서 우선순위 확인
   - 다양한 조합으로 테스트
