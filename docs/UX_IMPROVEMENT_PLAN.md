# HeyZeno UX 개선 계획서

## 🎯 목표
- 가입 직후 "와 이거 좋다"가 10초 안에 오도록 홈 강화
- 추천이 광고처럼 안 보이게 "근거/투명성"을 UI에 기본 탑재
- CTA를 줄여서 사용자가 고민하지 않게 (각 화면 Primary CTA 1개)
- MVP에서 비어 보이는 화면은 "Coming soon"이 아니라 "왜 필요한지"를 설계로 해결
- 텍스트의 주어를 "사료/가격"이 아니라 "{펫이름}"으로 바꿔서 감정 연결 강화

---

## 1) 화면별 냉정 리뷰 + 개선안

### 📱 /home (홈 화면)

#### 문제점
1. **로딩 중 "분석 중..."만 보여서 신뢰도 낮음** - 사용자가 "뭘 분석하는지" 모름
2. **추천 카드에 "왜 이게 추천인지" 근거 없음** - 광고처럼 보일 위험
3. **CTA가 2개 ("프로필 수정" + "맞춤 사료 보러가기")** - 선택의 부담

#### 개선 원칙
1. **진행 힌트로 신뢰 구축**: 로딩 중에도 "무엇을 하고 있는지" 명확히
2. **투명성 = 신뢰**: 추천 근거를 항상 표시

#### 즉시 적용 가능한 변경 (Low)
1. 로딩 중 "진행 힌트 카드" 추가 (알레르기 제외 완료, 나이/체중 반영 완료 등)
2. 추천 카드에 "왜 추천?" tiny secondary 버튼 추가
3. "프로필 수정" 버튼을 텍스트 링크로 변경 (Secondary 버튼 제거)

#### 다음 스프린트 변경 (Mid)
1. Pull-to-refresh 시 "방금 업데이트됨" 토스트 추가
2. 추천 근거 상세 모달 구현

---

### 📱 /watch ("내 사료")

#### 문제점
1. **추적 이유가 없어서 "왜 이걸 추적하는지" 모름** - 광고처럼 보임
2. **알림 스위치 토글 시 피드백 없음** - 햅틱/토스트 없음
3. **가격 정보만 있어서 맥락 부족**

#### 개선 원칙
1. **근거 표시 = 투명성**: 각 추적 카드에 추적 이유 1-2줄 필수
2. **피드백 = 신뢰**: 모든 액션에 즉각적인 피드백

#### 즉시 적용 가능한 변경 (Low)
1. 각 추적 카드에 "추적 이유" 1-2줄 추가 (예: "• 장 건강 케어", "• 평균가 대비 안정적")
2. 알림 스위치 토글 시 햅틱 + 토스트 ("가격 내려가면 알려드릴게요")
3. 가격 옆에 "최근 14일 평균" 표시

#### 다음 스프린트 변경 (Mid)
1. 추적 이유 편집 기능 추가
2. 가격 히스토리 미니 차트 추가

---

### 📱 /benefits (혜택)

#### 문제점
1. **MVP 초기 포인트가 0P로 비어 보임** - 이탈 위험
2. **적립 내역이 없으면 빈 화면** - "왜 이 탭이 있는지" 모름
3. **"혜택 안내"만으로는 동기 부여 부족**

#### 개선 원칙
1. **미션형 = 참여 유도**: 포인트 적립을 "미션 완료"로 전환
2. **예고 = 기대감**: MVP에서도 "곧 시작" 느낌

#### 즉시 적용 가능한 변경 (Low)
1. **A안 선택: 미션형 혜택으로 전환**
   - "알림 설정하면 100P" 미션 카드
   - "첫 추천 확인하면 50P" 미션 카드
   - "프로필 완성하면 200P" 미션 카드
2. 포인트 섹션에 "곧 시작해요" 배지 추가
3. 적립 내역이 없을 때 "미션을 완료하면 포인트가 쌓여요" 메시지

#### 다음 스프린트 변경 (Mid)
1. 미션 완료 애니메이션 추가
2. 포인트 사용 기능 구현

**A안 선택 근거:**
- 유저 심리: "0P"는 실패감, "미션"은 게임화로 재미
- 이탈 관점: 빈 화면보다 "할 일"이 있으면 재방문 유도
- 화면 구성: 카드 3개면 화면이 채워져서 비어 보이지 않음

### 📱 /me (마이)

#### 문제점
1. **"반려동물 프로필"이 데이터 나열만** - 돌봄 느낌 부족
2. **프로필 정보가 "견종/체중/나이"만** - 건강 포인트 요약 없음
3. **"프로필 수정" 버튼 위치 불명확**

#### 개선 원칙
1. **아이 리포트 = 감정 연결**: 프로필을 "아이 리포트" 느낌으로
2. **건강 요약 = 가치**: 건강 포인트를 1줄로 요약

#### 즉시 적용 가능한 변경 (Low)
1. 프로필 카드 헤더를 "뽀삐의 건강 리포트"로 변경
2. 건강 포인트 요약 1줄 추가 (예: "특이사항 없음, 건강해요!")
3. "프로필 수정" 버튼을 카드 하단으로 이동

#### 다음 스프린트 변경 (Mid)
1. 건강 리포트 상세 화면 추가
2. 프로필 사진 추가

---

### 📱 /products/:id (상품 상세)

#### 문제점
1. **"평균 가격 대비 최저가"가 추상적** - 구체적 근거 부족
2. **"가격 알림 받기" 버튼만** - 펫 이름 주어 없음
3. **상품 이미지 플레이스홀더** - 신뢰도 낮음

#### 개선 원칙
1. **구체성 = 신뢰**: "평균 가격"을 "최근 14일 평균"으로 구체화
2. **펫 이름 주어 = 감정 연결**: 모든 문구에 펫 이름 포함

#### 즉시 적용 가능한 변경 (Low)
1. "평균 가격 대비 최저가" → "{펫이름}에게 좋은 가격이에요"
2. "가격 알림 받기" → "{펫이름} 사료 가격 알림 받기"
3. 가격 정보에 "최근 14일 평균: XX원" 추가

#### 다음 스프린트 변경 (Mid)
1. 가격 히스토리 차트 추가
2. 상품 이미지 실제 데이터 연동

---

### 📱 /alerts (알림)

#### 문제점
1. **알림 메시지가 "평균가 대비 12% 하락"만** - 펫 이름 주어 없음
2. **알림 시간이 "2시간 전"만** - 맥락 부족
3. **알림 클릭 시 액션 없음**

#### 개선 원칙
1. **펫 이름 주어 = 개인화**: 모든 알림에 펫 이름 포함
2. **액션 = 전환**: 알림 클릭 시 상품 상세로 이동

#### 즉시 적용 가능한 변경 (Low)
1. 알림 메시지를 "{펫이름} 사료 가격이 내려갔어요"로 변경
2. 알림 클릭 시 상품 상세로 이동
3. 알림 시간 옆에 "지금 확인하기" 버튼 추가

#### 다음 스프린트 변경 (Mid)
1. 알림 필터링 기능 추가
2. 알림 설정 화면 추가

```
│ AppBar: "오늘, {펫이름}에게 딱 맞는 사료 🐾" │

│                                     │
│ ┌───────────────────────────────┐  │
│ │ 내 아이 카드                    │  │
│ │ {펫이름} · {나이} · {몸무게}kg  │  │
│ │ 건강 포인트: {요약}             │  │
│ │ [프로필 수정] (텍스트 링크)      │  │
│ └───────────────────────────────┘  │
│                                     │
│ ┌───────────────────────────────┐  │
│ │ 진행 힌트 카드 (로딩 중일 때만)  │  │
│ │ ✔ 알레르기 제외 완료            │  │
│ │ ✔ 나이/체중 반영 완료           │  │
│ │ ⏳ 최저가 트래킹 준비 중         │  │
│ └───────────────────────────────┘  │
│                                     │
│ ┌───────────────────────────────┐  │
│ │ 추천 Top1 카드                  │  │
│ │ {브랜드명} {상품명}              │  │
│ │ {가격}원 {변동률}%              │  │
│ │ [왜 추천?] (tiny secondary)     │  │
│ └───────────────────────────────┘  │
│                                     │
│ ┌───────────────────────────────┐  │
│ │ [Primary CTA]                  │  │
│ │ "{펫이름} 맞춤 사료 보러가기"    │  │
│ └───────────────────────────────┘  │
│                                     │
└─────────────────────────────────────┘
```

### Pull-to-refresh UX
- 상단에 "방금 업데이트됨" 토스트 (2초 후 자동 사라짐)
- 햅틱 피드백

### Primary CTA 문구 A/B 테스트 3안
1. **안1 (감정 강조)**: "{펫이름} 맞춤 사료 보러가기"
2. **안2 (액션 강조)**: "{펫이름}에게 딱 맞는 사료 찾기"
3. **안3 (간결)**: "{펫이름} 사료 추천 보기"

**추천: 안1** - "맞춤"이라는 키워드로 개인화 강조

## 5) /me 탭 "돌봄 느낌" 강화

### 프로필 카드 헤더 2안

**안1 (따뜻함)**: "{펫이름}의 건강 리포트"
**안2 (친근함)**: "{펫이름}이 건강해요!"

**추천: 안1** - "리포트"라는 단어로 전문성 + 돌봄 느낌

### 건강 포인트 요약 예시
- "특이사항 없음, 건강해요!"
- "알레르기 주의, 꼼꼼히 챙겨요"
- "장 건강 케어 중이에요"

### 프로필 수정 버튼 위치
- 카드 하단, 텍스트 링크로 변경 (Secondary 버튼 제거)

## 9) 카피 문구 세트 (바로 붙여넣기)

### 홈 화면
- AppBar: "오늘, {펫이름}에게 딱 맞는 사료 🐾"
- 진행 힌트: "✔ 알레르기 제외 완료", "✔ 나이/체중 반영 완료", "⏳ 최저가 트래킹 준비 중"
- 추천 카드: "{브랜드명} {상품명}", "{가격}원 {변동률}%", "[왜 추천?]"
- Primary CTA: "{펫이름} 맞춤 사료 보러가기"
- 프로필 수정: "프로필 수정" (텍스트 링크)
- Pull-to-refresh: "방금 업데이트됨"

### /watch 화면
- 추적 이유: "• 장 건강 케어", "• 평균가 대비 안정적"
- 알림 토글 토스트: "가격 내려가면 알려드릴게요"

### /benefits 화면
- 포인트 섹션: "내 포인트", "0 P", "곧 시작해요 🎁"
- 미션 카드: "알림 설정하기", "완료하면 100P 적립", "[시작하기]"
- 적립 내역 없을 때: "미션을 완료하면 포인트가 쌓여요"

### /me 화면
- 프로필 헤더: "{펫이름}의 건강 리포트"
- 건강 포인트: "특이사항 없음, 건강해요!"

### /products/:id 화면
- 가격 정보: "{펫이름}에게 좋은 가격이에요", "최근 14일 평균: XX원"
- 알림 버튼: "{펫이름} 사료 가격 알림 받기"

### /alerts 화면
- 알림 메시지: "{펫이름} 사료 가격이 내려갔어요"
- 확인 버튼: "지금 확인하기"
